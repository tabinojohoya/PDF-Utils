# PDF-Utils

macOS Tahoe (26.x) 向けのシンプルな PDF 結合・分割アプリです。複数の PDF を追加し、並び順どおりに 1 つの PDF に結合して保存できます。ページ単位の選択・除外・並び替え・ファイル間移動にも対応しています。

## Features

- **PDF 結合**: 複数 PDF を並び順どおりに 1 つのファイルに結合
- **PDF 分割**: 固定ページ数 / 指定範囲 / 1ページずつの 3 方式で分割
- **PDF 追加**: ファイル選択（複数選択）/ ドラッグ&ドロップ
- **ファイル並び替え**: リストのドラッグ&ドロップ / VoiceOver の「上に移動」「下に移動」アクション
- **ページ単位編集**: 各ページのサムネイルグリッド表示、個別ページの除外・並び替え
- **ファイル間ページ移動**: ページビューでページを別ファイルのセクションにドラッグ&ドロップで移動
- **プレビュー**: 右ペインで PDF プレビュー（PDFKit）、ページクリックで連動表示
- **プレビューモード切替**: singlePage ↔ continuous をワンクリックで切替
- **出力プレビュー**: 結合完了後、結果 PDF をアプリ内で即座に確認
- **結合**: 保存先を選択して結合（進捗表示・完了バナー・Finder で表示）
- **エラー処理**: 破損 PDF / 非 PDF / パスワード保護 PDF を検出して通知

## Requirements

- macOS 26.2+
- Xcode 26.2+

## Build

Releaseからdmgをダウンロードし実行してください。

## Usage

### 結合モード

1. PDF を追加
   - 画面にドラッグ&ドロップ、またはツールバーの「ファイル追加」
2. 必要なら並び替え（上から順に結合されます）
3. ページ単位で編集したい場合は `⌘2` でページビューに切替
   - 不要なページのチェックを外して除外
   - ドラッグ&ドロップでページ順を変更
   - ファイル間でページを移動（別セクションにドロップ）
4. 「結合」を押して保存先を選択
5. 結合結果が右ペインに自動表示 → 「編集に戻る」で再編集可能

### 分割モード

1. ツールバーで分割モードに切替
2. 分割したい PDF を選択
3. 分割方式を選択（固定ページ数 / 指定範囲 / 1ページずつ）
4. 「分割」を押して保存先フォルダを選択

### Keyboard shortcuts

| Shortcut | Action |
|----------|--------|
| `⌘O` | ファイル追加 |
| `⌘S` | 結合 / 分割を実行 |
| `⌘⌫` | 選択削除 |
| `⌘1` | ファイルビュー（結合モード内） |
| `⌘2` | ページビュー（結合モード内） |

## Sandbox / Permissions

本アプリは App Sandbox を有効化しています。

- 入力 PDF: ユーザーが選択したファイルにのみアクセスします（セキュリティスコープ URL）。
- 出力 PDF: `NSSavePanel` によるユーザー選択保存を行います。

## Known limitations

- パスワード保護 PDF（ロックされた PDF）は読み込み/結合できません。
- 非常に大きい PDF の結合は時間がかかる場合があります（進捗は表示されます）。
