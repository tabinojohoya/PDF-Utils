## ✨ New Features

### Sparkle 自動アップデート機能
- アプリ起動時に新バージョンを自動チェック
- メニューバー「PDF-Utils」→「アップデートを確認…」で手動チェック可能
- 新バージョンがあればダイアログからワンクリックで更新

### About メニュー
- 「PDF-Utilsについて」メニューで現在のバージョン・ビルド番号を確認可能

## 🔧 Internal
- Sparkle 2.x を Swift Package Manager で統合
- App Sandbox にネットワーククライアント権限を追加
- GitHub Pages 経由での appcast.xml 配信基盤を構築
- `scripts/release.sh` リリース自動化スクリプトを追加
- `.gitignore` を整備（ビルド成果物の追跡を解除）

## 📦 Install
DMG をダウンロードして、PDF-Utils.app を Applications にドラッグ&ドロップしてください。
